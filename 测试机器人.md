## 飞书文本格式说明

### 在飞书中支持的格式：

1. **粗体** - 使用 `**文字**` 或 `__文字__`
2. *斜体* - 使用 `*文字*` 或 `_文字_`
3. ~~删除线~~ - 使用 `~~文字~~`
4. `行内代码` - 使用 `` `代码` ``
5. [链接文字](https://example.com)

### 代码块示例：

```python
# 这是代码块
def test_robot():
    print("测试机器人")
```

## 如何在飞书中使用

飞书机器人发送消息时，建议使用：
- **Markdown格式** - 基础文本样式
- **消息卡片** - 支持更丰富的颜色和交互
- **富文本格式** - 通过API发送带颜色的文本
